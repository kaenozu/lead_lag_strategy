## 🎯 戦略の概要

### 仕組み（やさしく説明）

1. **米国の業種 ETF**の動きを見る
2. **数日前の米国市場**のパターンを分析
3. **日本の業種 ETF**がどう動くか予測
4. 上がりそうな銘柄を**買い**、下がりそうな銘柄を**売り**

### 特徴

- **ホライズン**: デイトレード（朝買って夕方売る）
- **取引頻度**: 毎日
- **必要資金**: 100 万円〜（推奨）
- **目標リターン**: 年率 5-10%
- **目標ドローダウン**: -15% 以内

## 🚀 始め方

### ステップ 1: 必要なものを準備

1. **証券口座**
   - 日本株 ETF が取引できること
   - 信用取引（空売り）ができること
   - 推奨：SBI 証券、楽天証券など

2. **資金**
   - 最低：50 万円
   - 推奨：100 万円〜
   - 余剰資金で！

3. **PC 環境**
   - Node.js がインストールできること
   - インターネット接続

### ステップ 2: セットアップ

```bash
# 1. プロジェクトディレクトリに移動
cd c:\gemini-desktop\lead_lag_strategy

# 2. 依存パッケージをインストール
npm install

# 3. データ取得（初回のみ）
node backtest_improved.js
```

### ステップ 3: 毎日の運用

#### 朝（8:00-9:00）

```bash
# シグナルを生成
node generate_signal.js
```

出力例：
```
📊 買い銘柄（ロング）
ランク  ティッカー  業種           投資額      シグナル値
  1      1618.T     エネルギー資源   100,000 円  17.70
  2      1632.T     証券・商品       100,000 円  8.74
  ...

📉 売り銘柄（ショート）
ランク  ティッカー  業種           投資額      シグナル値
  1      1621.T     医薬品          100,000 円  -276.80
  ...
```

#### 取引実行（9:00-9:30）

1. **買い注文**: 表示された銘柄を指定金額分買う
2. **売り注文**: 表示された銘柄を空売り

#### 夕方（15:00-15:30）

1. **決済**: すべてのポジションを閉じる
2. **記録**: 損益を記録する

## 📊 シグナルの見方

### 出力ファイル

| ファイル | 内容 |
|----------|------|
| `results/signal.json` | 完全なシグナルデータ（JSON） |
| `results/signal_long.csv` | 買い銘柄一覧（CSV） |
| `results/signal_short.csv` | 売り銘柄一覧（CSV） |

### シグナル値の意味

- **大きい正の値**: 強く買い推奨
- **大きい負の値**: 強く売り推奨
- **0 に近い**: 方向性が不明確

## 💰 資金管理

### 推奨ポジションサイズ

| 総資金 | 1 銘柄あたり | 銘柄数 | 総エクスポージャー |
|--------|-------------|--------|-------------------|
| 50 万円 | 5 万円 | 10 銘柄 | 50 万円 |
| 100 万円 | 10 万円 | 10 銘柄 | 100 万円 |
| 300 万円 | 30 万円 | 10 銘柄 | 300 万円 |

### リスク管理ルール

1. **1 銘柄 10% ルール**: 総資金の 10% を超えない
2. **ストップロス**: -5% で損切り
3. **ドローダウン制限**: -10% になったら一時停止
