## 🚀 クイックスタート

### 初心者向け（推奨）

```bash
# 1. セットアップ（初回のみ）
cd c:\gemini-desktop\lead_lag_strategy
npm install

# 2. 毎日のシグナル取得
node generate_signal.js

# 3. シグナルを確認して取引
# results/signal.json を参照
```

### 上級者向け

```bash
# バックテスト実行（データ取得から）
node backtest_improved.js

# リスク管理付きバックテスト
node backtest_risk_managed.js

# ペーパートレード開始
node paper_trading.js
```
