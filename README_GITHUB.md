## 日米業種リードラグ戦略

**初心者向け ETF 投資システム** - 楽天証券で 1 口から購入可能

## 🎯 概要

米国市場の業種 ETF リターンから日本市場の業種 ETF リターンを予測し、最適な買い銘柄を自動選択します。

### 特徴

- ✅ **楽天証券対応**: ETF が 1 口（約 3-9 万円）から購入可能
- ✅ **自動シグナル生成**: ページ読み込み時に最新シグナルを自動表示
- ✅ **初心者向け UI**: 買い候補のみを表示、投資額の目安も表示
- ✅ **パラメータ永続化**: 設定はブラウザに保存
- ✅ **CSV エクスポート**: シグナルを保存可能

## 🚀 クイックスタート

### 事前準備

```bash
# Node.js v18 以上が必要
node -v
```

### インストール

```bash
# 依存関係のインストール
npm install

# サーバー起動
npm run server
```

### 使い方

1. ブラウザで `http://localhost:3000` にアクセス
2. 自動的にシグナルが生成されます
3. 「買い候補」タブの銘柄を購入
4. 投資額を設定して再計算可能

## 📊 推奨投資額

| 銘柄数 | 必要資金目安 |
|--------|-------------|
| 1 銘柄 | 約 3-9 万円 |
| 5 銘柄（買い候補） | 約 25-30 万円 |
| 17 銘柄（全銘柄） | 約 80-100 万円 |

※楽天証券では ETF を 1 口から購入可能です

## 📁 主要ファイル

| ファイル | 説明 |
|----------|------|
| `src/server.js` | Web サーバー + シグナル生成 API |
| `public/index.html` | フロントエンド UI |

| `src/generate_signal.js` | シグナル生成 CLI ツール |
| `scripts/paper_trading.js` | ペーパートレードツール |

## 🔧 設定

### パラメータ

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `windowLength` | 60 | 過去何日間のデータを使用 |
| `lambdaReg` | 0.9 | 正則化強度（0-1） |
| `quantile` | 0.3 | 上位・下位何％を取引 |

### 環境変数（任意）

```bash
# .env ファイルを作成
PORT=3000
```
