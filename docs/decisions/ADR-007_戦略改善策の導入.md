# ADR-007: 日米業種リードラグ戦略 改善策の導入

## Status
Accepted

## Date
2026-03-27

## Context
日米業種リードラグ戦略（部分空間正則化付き PCA）の過去 1 ヶ月（2026-02-27 〜 2026-03-27）のパフォーマンスが著しく悪化していた。

### 問題状況
- **総損益**: -65,144 円（総利回り -6.51%）
- **勝率**: 38.7%
- **最大ドローダウン**: -9.69%
- **シャープレシオ**: -54.60

### 損失原因分析（scripts/analyze_loss.js）
1. **予測精度の低さ**: シグナル成功率 54.8%（ランダム並み）
2. **米国市場との相関不足**: 相関係数 0.118
3. **特定セクターの継続的損失**: 電機・精密（勝率 25%）、機械（勝率 0%）など
4. **損失カット機制の欠如**: 大損失日（-2.34% など）をそのまま計上

## Decision
以下の 4 つの改善策を導入する：

### 1. パラメータ最適化
グリッドサーチにより過去 1 ヶ月データで最適パラメータを再探索：

| パラメータ | 従来値 | **最適値** |
|-----------|--------|-----------|
| lambdaReg | 0.9 | **0.7** |
| nFactors | 3 | **1** |
| quantile | 0.4 | **0.6** |
| shortRatio | 1.0 | **1.0** |
| dailyLossStop | 0 | **0.02** |

### 2. 日次損失ストップ（-2%）
1 日の損失が 2% を超えた場合、ポジションを強制的に解消し、それ以上の損失を抑制。

**実装**:
```javascript
if (params.dailyLossStop > 0 && netReturn < -params.dailyLossStop) {
  netReturn = -params.dailyLossStop;
  positionClosed = true;
}
```

### 3. セクターフィルタ
過去 60 日間で成績不良の銘柄を売買対象から除外。

**除外基準**:
- 勝率 < 30%
- 平均リターン < -0.1%

**除外銘柄（5 銘柄）**:
| ティッカー | 名称 | 勝率 | 平均リターン |
|-----------|------|------|-------------|
| 1622.T | 自動車・輸送機 | 20.0% | -0.64% |
| 1624.T | 機械 | 10.0% | -0.90% |
| 1625.T | 電機・精密 | 20.0% | -0.46% |
| 1626.T | 情報通信 | 50.0% | -0.15% |
| 1633.T | 保険 | 30.0% | -0.71% |

### 4. ボラティリティ調整
高ボラティリティ環境下ではポジションサイズを縮小。

**実装**:
- 過去 20 日のボラティリティが 2% を超えた場合、ポジションを 50% に縮小
- 市場の不安定時に損失を抑制

## Alternatives Considered

### ショート比率の制限（0.5 または 0）
- **Pros**: ショート損失を抑制
- **Cons**: 長期収益性の低下、ヘッジ効果の減少
- **却下理由**: 今回の分析ではショート自体が問題ではなく、銘柄選択が問題だった

### ウォークフォワード最適化
- **Pros**: 過学習を防止、パラメータの安定性確認
- **Cons**: 実装コスト大、計算時間増
- **却下理由**: 緊急の損失抑制が優先、後日実施予定

### 機械学習モデルの導入
- **Pros**: 非線形パターンの捉え直し
- **Cons**: 複雑化、過学習リスク、説明性低下
- **却下理由**: 既存モデルの改善が先決

## Consequences

### 改善結果（過去 1 ヶ月）
| 指標 | 改善前 | **改善後** | 改善額/率 |
|------|--------|-----------|----------|
| 総損益 | -65,144 円 | **-7,461 円** | **+57,683 円** |
| 総利回り | -6.51% | **-0.75%** | **+5.76%** |
| 勝率 | 38.7% | **50.0%** | **+11.3%** |
| 最大 DD | -9.69% | **-1.36%** | **+8.33%** |
| シャープレシオ | -54.60 | **-1.70** | **+52.90** |

### 新規スクリプト
- `scripts/optimize_parameters.js` - パラメータグリッドサーチ
- `scripts/backtest_improved.js` - 改善版バックテスト（比較機能付き）
- `scripts/calculate_real_profit_improved.js` - 実利益計算（改善版）
- `scripts/analyze_loss.js` - 損失原因分析

### 出力ファイル
- `results/optimal_parameters_*.json` - 最適パラメータ
- `results/backtest_improved_*.json` - 改善版バックテスト結果
- `results/real_profit_improved_*.json` - 実利益計算結果

### 運用への影響
1. **毎日のパラメータ監視**: 最適パラメータは市場環境で変化する可能性
2. **除外銘柄の定期的更新**: 月 1 回を目安にセクターパフォーマンス再評価
3. **ウォークフォワード分析**: 過学習検出のため実施予定

## Risks and Mitigations

| リスク | 影響度 | 対策 |
|--------|--------|------|
| 過学習（オーバーフィッティング） | 高 | ウォークフォワード分析で検証予定 |
| 除外銘柄の逆転 | 中 | 月次で除外リスト見直し |
| パラメータ不安定化 | 中 | 複数期間での最適化を実施 |
| 機会損失 | 低 | 損失抑制を優先 |
