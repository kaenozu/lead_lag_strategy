初心者向け ETF 投資システム

⚠️ デモページについて
このページは静的なデモページです。実際のシグナル生成にはバックエンドサーバーが必要です。
✅ 主な機能
  • 楽天証券対応（ETF が 1 口から購入可能）
  • 自動シグナル生成（ページ読み込み時）
  • 投資額計算・表示（価格データ付き）
  • パラメータ永続化（localStorage）
  • CSV エクスポート機能
  • 初心者向け UI（買い候補のみ表示）

🚀 クイックスタート

  1. リポジトリをクローン
    2. git clone https://github.com/kaenozu/lead_lag_strategy.git
cd lead_lag_strategy
  2. 依存関係をインストール
    3. npm install
  3. サーバーを起動
    4. node server.js
  4. ブラウザでアクセス
    5. http://localhost:3000
