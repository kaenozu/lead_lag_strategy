# 日米業種リードラグ戦略 - 取引実行計画

> **作成日**: 2026年3月21日  
> **目的**: 学習重視の取引実践を通じて戦略の実効性を検証

---

## 状況整理

| 項目 | 状況 |
|------|------|
| 投資経験 | 中級者 |
| 利用可能資金 | 10-30万円 |
| リスク許容度 | 中程度（-10%まで） |
| 学習目的 | 系統的戦略の習得 |
| Altri資産 | 別途確保済み |

---

## フェーズ1: ペーパー貿易（1-2ヶ月）

### 目標
- シグナル通りに仮想資金30万円で取引
-  реаль市場と同じ条件で検証
- シグナル精度を把握

### 内容
1. **記録システム構築**
   - 日次シグナル記録
   - 仮想ポジション管理
   - 損益計算

2. **週次検証**
   - 週次パフォーマンス確認
   - シグナル精度分析
   - 問題点の特定

3. **成功基準**
   - 4週間以上継続
   - R/R > 0.3 維持
   - 最大DD < -10%

### 次のフェーズに進む条件
- ペーパー貿易で R/R > 0.3
- 2ヶ月以上運用経験
- 最大DD < -10%

---

## フェーズ2: 小口 реаль取引（2-3ヶ月）

### 目標
- 実際の5万円で取引開始
- ペーパーとの差異を確認
- スリッページ・手数料の影響把握

### 内容
1. **初期設定**
   - 軍資金: 50,000円
   - 1銘柄上限: 5,000円
   - 最大 同時持仓: 5銘柄

2. **取引ルール**
   - シグナル上位30%のみ買い
   - 寄り付き 成行注文
   - 引け 成行決済（当日完結）

3. **記録**
   - 全取引をCSV記録
   - 手数料・スリッページ込みの実損益
   - 週次サマリー

4. **成功基準**
   - 3ヶ月後 資金 > 50,000円
   - 月次 R/R > 0.3
   - 最大DD < -15%

### 次のフェーズに進む条件
- 3ヶ月以上継続
- 月次 平均リターン > 0
- 最大DD < -15%

---

## フェーズ3: 継続的改善（3ヶ月目以降）

### 月次レビュープロセス

```
每月末に以下を実施:
1. 今月の取引結果を集計
2. 勝率・平均損益・最大DD計算
3. 問題点があれば原因分析
4. パラメータ調整の検討（必要に応じて）
5. 来月の目標設定
```

### 評価指標（月次）

| 指標 | 目標 | 警戒線 |
|------|------|--------|
| リターン | > 0% | < -5% |
| R/R比 | > 0.3 | < 0.1 |
| 最大DD | < -10% | > -15% |
| 勝率 | > 45% | < 40% |

### 取引中止条件
- 月次リターン < -15%
- 3ヶ月連続赤字
- R/R < 0 持续

---

## リスク管理ルール

### 基本ルール
1. **1銘柄投資上限**: 総資金の10%
2. **総投資上限**: 総資金の60%
3. **損切り**: -5%で自動損切り
4. **日次撤退**: -3%で当日取引停止

### 禁止事項
- ナンピン（追加購入で平均単価を下げる）
- 週末持仓
- 感情的な取引判断
